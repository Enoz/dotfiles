# dotfiles

## nvim

Dependencies

- ripgrep
- fzf
- fd
- wl-clipboard (for c&p in wayland using `"+y` to yank into the `+` (clipboard) register)

```bash
ln -s -f ~/dotfiles/nvim/ ~/.config/nvim
```

## kitty

```bash
ln -s -f ~/dotfiles/kitty ~/.config/kitty
```

## ghostty

```bash
ln -s -f ~/dotfiles/ghostty ~/.config/ghostty
```
